# AI-Augmented SDLC Training Program

A hands-on Sprint 0 curriculum by [Stratpoint](https://stratpoint.com) for teams adopting AI-assisted software development workflows.

## What's Inside

| Folder | Description |
|--------|-------------|
| `presentation.html` | Full training presentation (open in browser) |
| `sample-project/` | Sample project artifacts and starter code used in exercises |
| `brd-agents/` | AI agent definitions for BRD generation |
| `ultimate-ai-agents/` | Organization-wide AI agent library (Claude, Cursor, Gemini, and more) |

## Topics Covered

- AI-Augmented requirements gathering and BRD generation
- Sprint 0 planning with AI assistance
- Prompt engineering for software teams
- AI-assisted code review, QA, and documentation
- Responsible AI use in the SDLC

## Getting Started

Open `presentation.html` in your browser to follow the training flow. Each module includes exercises using the materials in `sample-project/`.
