# Templates

> Reusable templates for training deliverables and gate reviews.
> Cohorts copy these into their sample-project repo and fill them in.

## Available Templates

| File | Purpose | Used at |
|---|---|---|
| `gate-signoff-template.md` | Gate review and sign-off form. One per phase gate. | Friday gate reviews (Week 1, 2, 3) |

## How to Use

1. Copy the template into your cohort's sample-project repo under `docs/reviews/`.
2. Rename for the gate: `week1-friday-signoff.md`, `week2-friday-signoff.md`, etc.
3. Fill in during the Friday gate review session.
4. Commit. The signed-off form becomes the formal handoff record.

## Adding New Templates

If your cohort builds a useful template (e.g., role-specific review form, traceability matrix template), contribute it back to this folder so the next cohort starts ahead.
